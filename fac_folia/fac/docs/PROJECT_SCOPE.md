# PROJECT_SCOPE

FAC (Fatsan Anti Cheat) targets Minecraft Java 1.21.11 with **Folia-first** execution discipline.

## Goals
- Packet-level core design.
- Low false positive, high maintainability.
- Lightweight runtime with staged-cost checks.
- ViaVersion-aware profiling.
- Staff-focused evidence and alerting.

## Non-goals
- “100% bypass-proof” claims.
- ML-only punishment.
- Heavy all-tick brute-force analysis.
